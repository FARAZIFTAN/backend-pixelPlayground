  • JWT token authentication
  • Session management & token refresh

2. User Profile Management CORE

  • View profile pribadi
  • Edit profile (name, phone, profile picture)
  • Change password
  • Upload & crop profile picture
  • View login history (last_login tracking)
  • Deactivate account
  • Delete account (soft delete)

  • Sort by: newest, most popular, most used

C. MODUL PHOTO BOOTH

6. Photo Capture Session MVP

  • Select template dari gallery
  • Start photo session with selected template
  • Access device camera (front/back)
  • Real-time camera preview
  • Countdown timer before capture (3-2-1)
  • Capture multiple photos (2, 3, or 4 based on template)
  • Retake individual photo
  • Progress indicator (Photo 1 of 4)
  • Apply filters to photos (optional)
  • Mirror/flip camera preview

7. Photo Composition MVP

  • Automatically composite photos with template
  • Real-time preview of final result
  • Adjust photo positions if needed
  • Apply filters to final composite
  • Add text overlay (optional)
  • Crop & rotate individual photos
  • Generate high-resolution output (for printing)

8. Photo Output & Sharing MVP

  • Download final composite as PNG/JPEG
  • Generate shareable link
  • Generate QR code for easy sharing
  • Share to social media (Facebook, Instagram, Twitter)
  • Send via WhatsApp
  • Email photo to user
  • Print-ready resolution (300 DPI)
  • Track download count
  • Track share count
